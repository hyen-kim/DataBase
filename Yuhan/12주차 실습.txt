< 10주 실습 풀이 (테이블변경) >

테이블 변경 : ALTER TABLE

1) 칼럼변경 		- 추가 : ADD
			- 수정 : MODIFY
			- 삭제 : DROP COLUMN

2) 제약조건 변경 	- 추가 : ADD CONSTRAINT
			- 삭제 : DROP CONSTRAINT
			- 변경 : 삭제 + 추가		--> 원래있던 제약조건 삭제 후 추가
